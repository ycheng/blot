分享終端機資訊

想要把在 Terminal 輸入的資訊 Share 給使用者看, 該怎麼做呢?

從 GWN 裡, 學到了使用以下的指令, 來分享 Terminal Session 給其他使用者看.

主控端:
mkfifo demo; script -f demo

使用者端:
cat demo

然後, 在主控端就給他開始打字吧! 是不是很好玩呢?

這個作法可以應用在為客戶做 Services 時, 請客戶在彼端看我們操作的步驟, 突然有個想法, 如果把 cat demo 這一段寫程 java code, 是不是可以透過 web 端來看, 也就讓客戶會更方便囉! 真是讚!
