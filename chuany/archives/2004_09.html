September 12, 2004

vim syntax on

Debian 在文字模式下預設使用 vim 時, 並沒有把 syntax 啟動, 所以會沒有漂亮的彩色模版出現, 您可以在使用家目錄下建立一個 .vimrc 的檔案, 內容加入 syntax on, 這樣子下次再進入 vim 後, 就可以輕鬆愉快的編輯囉!
