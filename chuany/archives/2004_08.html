Single Mode

一般而言, 我們所認知的 Linux Single Mode 都是不需要輸入任何密碼就可以進入系統, 然而 Debian GNU/Linux 就需要輸入密碼才可以, 要如何才不用輸入呢?

參考此URL:http://lists.debian.org/debian-user/2001/05/msg00492.html

LILO: linux init=/bin/sh
# mount -o remout,rw /

就這樣子簡單.

KVM - Synergy

在 IRC 的 #debian.tw 頻道上經由 julian9 分享出的 blog , 而得知這個有趣的軟體, 如果想使用軟體 KVM Solution, 可以參考看看.

http://darwin.nchc.org.tw/blog/archives/000032.html

簡單步驟:
Windows-(Server)
1.http://prdownloads.sourceforge.net/synergy2/synergy-1.0.6-1.zip?download
2.解壓縮後, Double click Synergy synergy.exe 圖示
3.選擇Server
4.選 Add 加入 windows 跟 linux 兩台主機的 'Screen Name'(一般用hostname)
5.在主畫面,選擇剛加入的 linux 'Screen Name'
6.在右邊的 Layout 裡, 把 'Left:' 後面的選單選擇 windows 的 'Screen Name'
7.選擇 windows 的 'Screen Name', 在右邊的 Layout 裡, 把 'Right:' 後面的選單選擇 linux 的 'Screen Name'
8.點選 Test , 若 okay, 再點選 Start.

Linux-Client
1.從 Windows ssh to Linux
2.Run /usr/bin/synergyc -f --no-camp windows.example.com
