預先安裝 Linux 的 Notebook

昨天下午的聚會, 我們再次談到預先安裝 Linux 的 Notebook. 這是個很痛的事,
因為花了大筆的銀子買的 Notebook, 當然希望可以發揮他完整的功能.

然而大部分的時候, 我們都需要既努力, 又辛苦的去確認哪一台 Notebook
對 Linux 有好的支援, 最後決定了以後, 在新版的系統出來以後, 有時運氣
不好還會遇到新版系統不支援的情形. 想到就令人鼻酸.

下面這個網址有在販賣預先安裝的 Linux.
https://www.qlilinuxpc.com/
看起來保固也 ok. 價錢普普, 不過就不清楚他們是否可以保證新版的 linux
出來, 驅動程式一定可以 work.

另一個賣 Toshiba 的 Notebook, 且提供預先安裝 Linux 的服務http://www.swt.com/notebooks.html

這一個網址包含許多人製作的網頁 Linux, 連結到支援 Linux 的 Notebook 網頁
http://www.linux-laptop.net/

IBM 也有一些跟 Linux 有關的網頁, 這邊有一些關於在 ThinkPad 上面使用 Linux
的簡單說明 : http://www-3.ibm.com/pc/support/site.wss/MIGR-48NT8D.html
