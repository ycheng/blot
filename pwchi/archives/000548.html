GnuPG 1.2.4 released

今天 GnuPG 1.2.4 Stable Version 發佈…

1.2.4 版主要有兩大變動部分：

移除所有對 Elgamal sign+encrypt keys (type 20) 的支援功能。
也就是說，自 GnuPG 1.2.4 起，所有的 type 20 金鑰，即無法被辨識和使用。

另外，新增了對 BZIP2 的壓縮演算法支援，不過目前尚在實驗階段，所以直提供讀取的功能，還無法產出經 BZIP2 壓縮的密文。

由於 GnuPG 1.2.3 版，存在 type 20 金鑰的安全問題，所以建議大家儘快更新版本。
