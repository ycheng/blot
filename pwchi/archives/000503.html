無奈的重回 Windows

無奈的一天 ～ 一個多月前的 8/23 我正式走向 Linux 的懷抱。一個多月後，礙於某些因素，我又回到了 Windows ...

突然間，有一種「又被關起來了」的感覺。

會跳回 Windows 主要有幾個原因，這些問題都是在之前評估時，沒有注意到的。
之前我都只注意到軟體方面的解決方案，彈卻忽略調硬體的問題。

1.我一直找不到我這片 BUFFALO IFC-USCB SCSI 卡的 Linux 的驅動程式。這讓我沒有辦法使用我的 MO 機。
2.我試了許多方法，始終無法讓 ACPI 起來。
3.我 NB 上的許多功能鍵，沒有辦法在 Linux 下正常作用。

我花了一點時間，希望理性的思考幾個問題。
我是否應該執著於作業系統的迷思或商品的最大利用率？

我想，使用 Windows 並不會影響我對自由軟體的熱誠。
