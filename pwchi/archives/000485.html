ThizLinux

朋友說，Debian 正在向我大喊，回頭是岸啊！！

搞了1/3天，Debian 的 X-Window 仍然宣告失敗。在我心煩之餘，把上次在慶生會拿 Dr.Linux 送的 ThizLinux 7.0 裝了上去！喔～～和 MDK 很像，有 RH 的影子！

至少 ThizLinux 7.0 一裝好，就有一個完好的 X-Window 等著我用。不過有個狀況，PPPoE/ADSL 連線設定好之後，是可以正常驗證並取得 IP 。但是就沒辦法上網，研究了半天，好像是 Route Table 的問題。調整一下之後，就可以上線了。

ThizLinux 7.0 基本安裝小於 1.8GB ，內建的 Browser 是 Mozilla 1.2.1 ，ThizLinux 也提供 On-Line Update 的服務！整體感覺，很適合初學者拿來當作 Desktop 用！
