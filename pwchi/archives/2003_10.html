無奈的重回 Windows

無奈的一天 ～ 一個多月前的 8/23 我正式走向 Linux 的懷抱。一個多月後，礙於某些因素，我又回到了 Windows ...

突然間，有一種「又被關起來了」的感覺。

會跳回 Windows 主要有幾個原因，這些問題都是在之前評估時，沒有注意到的。
之前我都只注意到軟體方面的解決方案，彈卻忽略調硬體的問題。

1.我一直找不到我這片 BUFFALO IFC-USCB SCSI 卡的 Linux 的驅動程式。這讓我沒有辦法使用我的 MO 機。
2.我試了許多方法，始終無法讓 ACPI 起來。
3.我 NB 上的許多功能鍵，沒有辦法在 Linux 下正常作用。

我花了一點時間，希望理性的思考幾個問題。
我是否應該執著於作業系統的迷思或商品的最大利用率？

我想，使用 Windows 並不會影響我對自由軟體的熱誠。

October 04, 2003

單面雙層的 DVD-9 燒錄規格

近期 "Pioneer" 與 "DVD+RW 聯盟" 都分別發布令人振奮的好消息 ...

DVD-RW 大廠 Pioneer 與 DVD+RW 組織，都在最近發表了 DVD-9 的單面雙層標準，兩者最大容量都可達 8.5 GB，且均相容現階段的 DVD-ROM 光碟機。

雖然已經有 4.7GB 的 DVD 燒錄機，但是許多 DVD 影片，均採用 DVD-9 的格式。想要合法備份的使用者，常常需要自行切割，分片，造成相當的麻煩。如今，新的 DVD-9 燒錄機，即將登場。相信這對許多人來說，是個相當興奮的好消息。

支援 DVD-9 的 DVD+RW 燒錄機，預計 2004 年上市!
