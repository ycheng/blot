從空中看地球

從空中俯看大地，那一份抽離的喜悅，寬闊的視野，空中攝影師 Yann Arthus-Bertrand 用他的鏡頭記錄了這份悸動，用攝影作品和我們分享這片共同生活的大地。

目前中正紀念堂外正在展出「從空中看地球—空中藝術攝影特展」。相當推薦各位朋友，利用閒暇之餘前往參觀。這次展出的所有作品，都是在直昇機上所拍攝的空拍照片。從空中以宏觀的角度看這世界，有著完全不同的感動。「從空中看地球」多半以「自然保育」與「人文關懷」為兩大主題，這展覽我看了兩次，一共展出一百五十幅左右的作品，每當重新再看同一幅畫時，總是會有不同的感受。是值得一看在看的展覽，推薦給大家，相信在欣賞的同時，你也會和我一樣，覺得地球是可貴的。

真的不妙

昨天把電池帶去光華找人檢查之後，結果看來是沒救了。不過聽到一個很悶的消息……

自從原廠代理商那得知，我這型電池一個要價七千多塊以後，心裡就一直不平衡。為什麼一個電池要七千多塊、為什麼一個七千多塊的東西只能用一年多。雖然在電池廠的同學告訴我，電池壽命得卻大概就一到兩年，要我節哀順變。但是上述的抱怨之聲，還是不時的在我腦海出現。

更悶的是，光華一間專搞電池的老闆說：「你這電池沒救了，他裡面的控制電路有設計在多久時間以後會自動壞掉。」我說你怎麼知道，老闆說：「看電池哪間代工廠就知道，那幾間代工廠的電池都會這樣搞。」

我不曉得這句話的真實性有多少，不過卻讓我很悶，有被坑的感覺。

GnuPG 1.2.4 released

今天 GnuPG 1.2.4 Stable Version 發佈…

1.2.4 版主要有兩大變動部分：

移除所有對 Elgamal sign+encrypt keys (type 20) 的支援功能。
也就是說，自 GnuPG 1.2.4 起，所有的 type 20 金鑰，即無法被辨識和使用。

另外，新增了對 BZIP2 的壓縮演算法支援，不過目前尚在實驗階段，所以直提供讀取的功能，還無法產出經 BZIP2 壓縮的密文。

由於 GnuPG 1.2.3 版，存在 type 20 金鑰的安全問題，所以建議大家儘快更新版本。

相容中文檔名的 GnuPG 前端軟體 - WinPT

最近發現，原本常用的 GPGShell 似乎對中文檔名頗感冒，每當要處理含中文檔名的加密檔案時，GPGShell 就常自爆！

這並不是 GnuPG 的問題，應該是 GPGShell 本身在處理中文檔名時的瑕疵。友人 Jedi 兄，曾向 GPGShell 開發者反映此一問題，但據說石沈大海毫無音訊。

忽然間，我對久未使用的 WinPT 燃起一絲希望，說不定換個 frontend 會有好消息。果然實測了一下，WinPT 在處理中文檔名上，似乎相當順利，目前尚未發現問題。

如果你需要處理中文檔名的話，相信 WinPT 會是較好的選擇。不過 WinPT 進展較為緩慢，功能也較無 GPGShell 豐富。需要的朋友可以比較一下各取所需。

December 18, 2003

GnuPG's ElGamal signing keys compromised

GnuPG.org 在 2003-11-27 緊急發佈一則 GnuPG 重大安全漏洞的警訊，請求使用者立即廢止有使用 ElGamal signing keys 的金鑰。

法國的一位研究員 Phong Nguyen 已經首先證實，GnuPG 在建立並使用 ElGamal Keys 簽署的過程中，存在一個嚴重的安全漏洞！這個嚴重的漏洞會導致 private key，在數秒的時間內被破解。當你的金鑰有使用 ElGamal sign+encrypt keys (type 20) 時，也就是你的 ElGamal 可同時用來加密和簽署，那這個問題就會存在。

由於 GnuPG >= 1.0.2 的版本才能製作 type 20 的金鑰。換句話說，由 GnuPG < 1.0.2 的版本所製作的金鑰並不會有這問題。

如何檢查是否為 type 20 金鑰：
1. 使用 gpg --list-keys 來檢視自己的金鑰資料。
$ gpg --list-keys xxxxxxxx

2. 如果你金鑰類型部分是大寫的 "G" ，例如下列 "2048G" ，那麼就表示這把金鑰是一支 type 20 的金鑰。你應該立即停用並發佈廢止通知。
pub 2048G/xxxxxxxx 2001-xx-xx Mallory

PS. 如果是小寫的 "g"，那這是一支標準的 ElGamal encrypt-only keys (type 16)。只能用來加密，並不會構成加密環節的威脅。

！！只有大寫的 "G" 必須立即廢止！！
