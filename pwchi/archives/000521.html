資訊書也可以這麼可愛

這本書如果不看書名，完全無法讓我聯想到是資訊書籍。但也因為這樣，或許更平易近人…

在映象中，電腦書的封面就是一副冰冷樣。不過這本書，改變了我的觀點。他讓我發現，原來電腦書也可以這樣搞！！
一看到封面就被吸引住目光了，瞬間還有點反應不過來，怎麼動畫書擺到這區來了。我不曉得這種風格的電腦圖書在日本是不是創新，不過我相信在台灣應該是首創。他讓電腦書不像一般的電腦書，變的更平易近人。

講到這裡，我唯一在天瓏看過這類型的中文書，是一本 JAVA 方面的入門書，它以插畫圖解的方式，解說 JAVA 與 OOP 的基礎知識，記得作者也是日本人，台灣有出中文版，一時忘記書名，我覺得是一本絕對適合初學者的入門書。下次有機再提出來和大家分享吧。

其實, 在小教授二號的時代就有了.

** 以下全憑記憶, 講錯不管 **

應該還有人記得張小燕主持的綜藝一百, 她沒事就在那裡 "拉拉拉, 我是易百拉~". 好, 重點不是這個. 重點是, 當時在節目中就已經有利用電腦動畫來表示排行榜的上升下降. 那個程式就是唐澤 (不太確定, 回家再查一下好了) 在小教授二號寫的.

** 回歸正題 **

這位仁兄也是遊戲 "撲克教授" 的作者. 這可以算是當時相當流行的大型電動 "金撲克" 的移殖版. 他也著了一本同名 (撲克教授) 的書, 從小教授二號的基本操作, 它上面的 Basic 語言, 文字模式, 高低解析度的繪圖模式, 一直到整個金撲克的構思, 撰寫. 書末還附上撲克教授的完整源碼 (台灣最早的開放源碼? *笑*). 而且... 幾乎每頁都有機器娃娃與怪博士的插圖 (他請朋友畫的), 這對當時只是國小的我, 真是愛不釋手. 這也是我從以前到現在, 唯一一本翻到書皮掉下來, 整本爛爛的書.

哪天有空來幫它寫個 blog 好了. :)
