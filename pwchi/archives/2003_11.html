資訊書也可以這麼可愛

這本書如果不看書名，完全無法讓我聯想到是資訊書籍。但也因為這樣，或許更平易近人…

在映象中，電腦書的封面就是一副冰冷樣。不過這本書，改變了我的觀點。他讓我發現，原來電腦書也可以這樣搞！！
一看到封面就被吸引住目光了，瞬間還有點反應不過來，怎麼動畫書擺到這區來了。我不曉得這種風格的電腦圖書在日本是不是創新，不過我相信在台灣應該是首創。他讓電腦書不像一般的電腦書，變的更平易近人。

講到這裡，我唯一在天瓏看過這類型的中文書，是一本 JAVA 方面的入門書，它以插畫圖解的方式，解說 JAVA 與 OOP 的基礎知識，記得作者也是日本人，台灣有出中文版，一時忘記書名，我覺得是一本絕對適合初學者的入門書。下次有機再提出來和大家分享吧。

November 21, 2003

英普達結束營運

在台灣經營 ISP 業務達十一年的"英普達"，由於財務危機宣布，於 11/17 日宣佈結束營運…

由於英普達的上網包，常常是電腦硬體的隨機搭售贈品，所以很多人用不到就拿到網路上拍賣。當初在拍賣網站，買到英普達的一年上網包，價錢相當便宜。記得，一年份只花 1200$ 元就買到，以現在的眼光，還是只有五折的超低價位。

在使用 IS.NET 這一年半的時間中，對 IS.NET 的感覺相當滿意，不管是連日本、大陸、香港、美國...etc.，網路速率都維持在水準之上，真的是便宜又大碗。

如今， IS.NET 宣佈結束營運，不禁感嘆，又一個好的 ISP 消失了。由衷感謝 IS.NET 這些時間來的服務。謝謝！

現在 IS.NET 國外路由已經斷線了，趁國內線路還沒斷線之際，趕緊物色新的 ISP 吧。

Linux 的發音？！

聽過的 "Linux" 唸法百百種，如果問我到底該怎麼念？那不如聽聽 Linux 的老爸 Linus Torvalds 他怎麼說…

最近有朋友問到，"Linux" 這個字到底該怎麼唸，因為聽過的版本實在太多。為了回答這個問題，特別重金請到 Linus Torvalds 來親自示範一段。:-P

內容為："Hello，This is Linus Torvalds and I pronounce Linux as Linux"

英語
瑞典語

其實 "Linux" 的發音，在每個地區，都可能由於當地口音的不同，而有些微的差異。因此，Linux 的發音，或許並沒有所謂官方的標準發音。目前的情況應該是，大家聽得懂就好。:P
