新主機上路感想

昨天剛拿到 CPU，馬上裝上 tyan-2466-4m，但不幸也開始了，重裝 windows XP 竟然一直出現藍白畫面，而且...賣主送的北極風真的是北風冷颼颼，吵死了啦！！！

忙了個一小時，決定睡覺不弄了，隔天不信邪，單顆或雙顆個別測試，一樣出問題...T_T...後來轉機來了，進到 Bios，reset Bios Default，才發現到原本在 Ram 的設定中是設定成 ECC Mode，因為我買的是 Registered ECC ram，或許是因為不相容吧，關閉 ECC 之後，重新開機，先用單顆安裝 WinXP，咦...很順耶....

接下來裝好兩顆，在重裝一次，哇....都正常了耶，讚讚讚，馬上進到 Linux 之中，開 Vmware 然後用Mozilla 到 flash plugin + Java 的網頁，天阿...Vmware 一直在執行，Mozilla 依然順到不行，damn good and stable...

實在不敢相信阿，這才叫做多工阿，目前還沒換 SCSI HD，到時候一定更順啦...未來一定也是買 Dual CPU 的...

只是...風扇太吵了....明天去找找看有無 12V 轉 5V 的接頭，變成一個安靜的雙 K7 主機...ccc
