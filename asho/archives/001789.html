disk on memory

之前想把自已用的小Server的OS裝在USB Disk上，昨天在公司完成後，確定可以開機了，興致高揚地帶回家，結果...家裡的主機板不支援usb disk開機，殘念...

於是，在什麼都有賣的地方找到了這個好玩的東西：Disk on memory

看到這個東西，實在是讓我興奮起來，查了一下CF卡的價格，1G大約700塊左右，實在是可以投資的東西。看來要來買一個用看看，我想有了這一個東西，不管是在lab裡面測試Linux還是當自用的Server當很方便吧。BTW...windows需要多少的容量才能安裝呢？

另外，我也找了外接板的，搭配epia應該就有很多應用可以做了。
