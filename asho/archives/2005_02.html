February 22, 2005

聯力 PC-71 的親密接觸

今天早上九點半，快遞就打電話給我，心想應該是 Case 送來了，簽單的時候，貨運大哥還說怎麼會這麼大一台阿...XD

拿到 Case 的時候，雖然箱子很大，但是重量很輕，一隻手就可以提起來整個箱子，參閱下圖箱子的大小。

如果不說螢幕是Philips 107P，還以為是 15寸的小螢幕呢。

規格寫 7KG，但感覺上卻沒那麼重，拆開後，馬上被 PC-71 酷炫的黑光射傷眼睛，全鋁鎂合金的機殼原來是長這樣子阿，真的是大開眼界了，側版，面板，floppy，CD-ROM都是全鋁的，難怪價格上也多了一截。

這個case真的很大，可以裝12x13的大型主機板，跟舊的Hec 69U9相比，PC-71算是巨人了吧...XD，底下的圖是case對照表

這是舊的case和PC-71的側身圖，您可以看到PC-71真的大很多，再跟一般四大的case比一下：

是否體會到他真的有夠"大"呢？...

接下來就是準備要開殼了，參照說明書拆開機殼後，發現左上角有損毀和些微斷裂，應該是出貨前就造成的...Orz...

接下來準備把小黑上得配備換到PC-71上頭，為了緬懷他之前的盡忠職守，特地為他照了最後的"遺照"：

您看看，真的很擠吧，SCSI線幾乎沒地方擺，硬碟幾乎撞到顯示卡，最重要的是很難整理電源線...拆下所有裝備之後，準備上 PC-71。

PC-71有很多貼心的地方，比如Power的位置是可以後移，直接把擋板拆掉，Power裝上之後再滑進Case裡面鎖好，以後還能擴充雙Power呢。硬碟擺放採用直立式（可裝六顆硬碟），有附轉接片變成橫躺的，但只能裝三個硬碟，Apple的機器都是直立的，加上看了很多文章，直立對硬碟沒有影響，所以全部硬碟都直立放就好了。

接下來是上主機板，背版要先裝上主機板的mounting nuts，這個東西阿，惡搞我一下...@@，原本我放進去的時候，只裝了半高，因為卡住了，我在想到底要不要用力擠進去，因為用力擠的話，怕會傷到這貴鬆鬆的機殼，所以暫時裝半高，但事實證明這是錯的。

安裝介面卡時，就發現插曹對不準，鎖不起來，鍵盤和滑鼠的插座也太低，所以主機板拔起來，把mounting nuts用尖嘴鉗小心翼翼地壓到底，總算解決了我的疑惑，介面卡和插座一切歸位。其實想一想，聯力賣這個貴的Case怎麼會犯這種糊塗事呢...

安裝好的圖片如下：

您看看，是不是空曠多了，雖然線還是很多，但至少位置變大了，不會卡在一起，讓機殼不通風，附帶說一下，前後各有兩個ADDA金色標籤的8x8風扇，ADDA的風扇可說是靜音型風扇且風量不算小喔，算是另一個欣喜的地方。只不過為了裝這個 Case，也把我累壞了...

完整組裝開機後，系統溫度下降好多喔，系統風扇和Power風扇吹出來的風是"涼"的，真恐怖阿...這是我第一次買聯力的Case，而且挑這麼大一顆，真的很滿意，難怪有人說用了聯力的case之後，就會變成忠實用戶，PC71除了非常大之外，真的要挑缺點的話，就是不能坐在上面，因為會"凍"傷...-_-|||...給想買這一款機殼的人參考看看...:-D

BTW...其實我是想要買一般Case高度的機殼就好，但能夠裝Extended ATX的主機板，上聯力網站笨笨地直接選大型機殼，難怪找不到普通高的，後來在case的DM上看到 PC-62...@@...差點昏倒，只不過也沒關係啦，我是以更便宜的價格買到 PC-71，要偷笑了...ccc，以後再買 PC-62 就可以了...聯力機殼棒阿...

Konsole 快速鍵

自從使用 KDE 當我的 WM 之後，我也開始使用 Konsole 當我的 default terminal 了。Konsole 有許多迷人的快速鍵，我常用的如下：

Ctrl + Alt + N：開啟一個新的 session
Ctrl + Alt + S：rename session name
Shift left/right：move to the left/right session

這些東西都可以在 快速鍵的地方找到，還有好用的 bookmark 功能，請按 F1 取得 konsole 的說明

KDE backports for woody

KDE 3.2.3 still has an available Woody backport:

deb ftp://ftp.kde.org/pub/kde/stable/3.2.3/Debian woody main

The source tarballs are still available in pool:

ftp://ftp.kde.org/pub/kde/stable/3.2.3/Debian/pool/main/

If you want a deb-src, you'll need to pull down the pull and make your
own with dpkg-scansources.


We use the 3.2.3 version above with a few bits of flakiness (kwin
becomes unresponsive on occasion, and certain control center panels
crash horribly).

I'm all for collaborating to provide a newer 3.3.2 backport. Heck, I'd
even be willing to help support it a bit, as we need to support it
in-house anyway.

- Ian C. Blenke
