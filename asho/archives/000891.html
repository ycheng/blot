Konsole 快速鍵

自從使用 KDE 當我的 WM 之後，我也開始使用 Konsole 當我的 default terminal 了。Konsole 有許多迷人的快速鍵，我常用的如下：

Ctrl + Alt + N：開啟一個新的 session
Ctrl + Alt + S：rename session name
Shift left/right：move to the left/right session

這些東西都可以在 快速鍵的地方找到，還有好用的 bookmark 功能，請按 F1 取得 konsole 的說明
