第二次了

今天早上發生一件慘案！就是小弟我錯把洗面乳當牙膏了.. >_< ...更慘的是，我還是在刷完牙之後才發現刷的是洗面乳....

這是我人生第二次幹這種蠢事，上一次是大學某次熬夜趕 C 作業後的傑作！

話說一早起來睡眼惺忪進了浴室，遍尋不著牙膏，腦筋閃過一個記憶：『 昨晚已經榨完牙膏先生的最後一吋白肉 』， 出了浴室翻開櫃子找了條我認為是牙膏的條狀物 ( 按：因為我眼鏡拿下來了，又兩條實在長太像，直覺認為那就是牙膏 ）， 擠了一陀沾在牙刷上就開始刷，『ㄟ...味道好像不太一樣，唷～冒泡了...應該沒錯吧！』 心裡想， 刷完之後，洗面乳的味道開始從我嘴裡散發出來，『 Oh My god .. 苦的味道這時候佈滿我的口腔』 ... 戴起眼鏡仔細一看：『挖ㄌㄟ，又幹一次蠢事了，大學時的那次記憶全湧上來了，本來昏睡的狀態，頓時精神百倍 .... Orz 』。

趕緊拿『真』的牙膏重新刷了幾次，洗臉的跟洗牙的雖然差一個字，但是味道可是南轅北轍，連散去的時間都差很多，真正沒味道的時候以經是中午吃過飯後了... Orz Orz...

好啦～提供這個慘痛經驗是希望大家不要以身試法，叔叔是有練過的，小朋友可別學唷.. >_<

很不錯的心智軟體 - FreeMind

有時候我們會把心智軟體叫做魚骨頭軟體，看一下上面這張圖，就不難理解為什麼叫魚骨頭了 ， 很多朋友可能還不太清楚心智軟體的作用為何？其實它非常的好用，應用的層面也非常的廣泛.....

心智軟體的一大功用在於思緒的整理，當我們遇到一些問題一時間不知道怎麼處理的時候，便可以透過這套軟體來整理一下思緒，使我們能夠找出最佳的解法。

每個人都會遇到的問題，不管是工作上或生活上或課業上，當問題產生的時候不要急著做出決定，先透過心智軟體來整理一下，透過軟體的輔助可以讓我們找到最佳的解法，舉個簡單的例子：

* 假設我是一個想要接觸 Linux 的人，我到底要選擇哪一套 Linux distribution 作為開始比較好呢？

透過這樣的過程，我們可以很清楚的瀏覽自己的思緒，依照自己需求的比重，尋找出最適合的答案！

上面只是一個小小的例子，人生還有很多困腦我們的問題！
* 對大學生來說，我該選什麼課？
* 對剛踏入社會的人來說，該先買車還是先買房子？

失敗的談車首航

唉..車子買不成還受一堆鳥氣...這是發生在這禮拜二的事情 ( 2005/1/18 )

當天去永和某福特名X營業所談 Focus 5D ...

小弟生平第一次買汽車，本著資訊人的職業病 ，談車前利用一兩天的時間上網惡補了汽車的一些基本知識，帶著一堆網路上蒐集到的資料，單槍匹馬前往營業所，下面文章提到的汽車配備原則上都是網路上大家比較推崇的品牌.說實話我也不是很懂 ( 汗 )


基本上我本來的主要期望是：

* 隔熱紙 - 前檔 V-Kool J60, 其他 3M 8803 ( 含天窗 )
* 第二隻遙控晶片鑰匙 ( 原廠會附有一把沒有控功能的鑰匙 ，我希望多一把遙控的 )
* 海瑪腳踏墊
* 鋼甲武士八代

次要期望：

* 避光墊
* 測速器
* 輪胎氮氣填充

其他小樣：

* 擋泥板
* 防滑墊

大致上期望是這樣...結果談的過程是...

*隔熱紙： J60 前擋沒辦法...要的話他可以找到 6000 元，真要的話要補差價.，然後我說其他部份應該有含天窗吧。他說：『什麼!!沒有人在天窗貼隔熱紙的拉...』，那網路上說有的朋友不是人嗎？!... >_<

*多一把遙控鎖..沒機會..他可以幫忙問價錢..要打的話就是自己付錢..

*海瑪腳踏墊：他願意! ( 真難得 ) ，但是通用的規格不能定做。

*方向盤鎖：業代說：『什麼鎖都沒有用拉，只是讓小偷看到減低偷車意願，那些功能都是雜誌在誇大的 ，我會送你一個方向盤鎖..就是兩頭...(我已經不想聽下去)。

* 氮氣部份：業代說：『唉唷!那個氮氣根本沒用..我上次怎樣怎樣.....( 我心理想：阿沒有的話幹嘛有這種產品...又是雜誌誇大 ? ) 』...所以當然氮氣這樣就沒下文...

* 避光墊： 我覺得這東西應該沒問題吧! 避光墊一說出口，馬上就跟我說：『那個是怕反光看不到儀錶板才需要的拉，我們 focus 是深色儀錶板，不需要拉』...老實說我還真不太清楚避光墊的功用...所以...我又閉嘴

* 其他小樣：唉...我實在不敢冀望還能談到什麼東西，本來列印出來的筆記就沒繼續念了.. -.-

殘念阿~~~ 東西凹不到...車價總得給我一點回報吧，起碼可憐我晚餐沒吃就衝去談車的誠意...

車價部份，就是跟我前面 post 那樣：

* 弄到好 ( 含領牌、牌照稅、燃料稅 ) 保乙式免自付額 72 萬
* 弄到好 ( 含領牌、牌照稅、燃料稅 ) 保丙式免自付額 70 萬

其實他講一堆保險的東西，這邊加過來那邊加過去，我是有聽沒有很懂，總之他說送我一份丙式保險加丟車賠車險，然後如果我還要保乘客險、第三責任險、竊盜的話：
* 乙式自己要再付： 34xxx
* 丙式自己要再付： 18xxx

講坦白話，依照我談到的配備，我真的不太曉得車價要多少是比較合理..但前面提什麼都被兔回來，我希望他起碼車價給點彈性，好歹我過程中都聽他說也沒有太多反駁...

結果當然是他很硬，我也硬，然後我覺得不能接受就說：『那我們就做個朋友好了』，話一出對方臉色大變說：『好!不要浪費時間』東西收一收就往營業所後面走去... 留下措扼不已的我...( 阿~~ 現在是怎樣... ) ，自己的東西收一收..我就也走了..

真是慘痛經驗阿，這故事告訴我們，要談車最好還是多人去，而且要找懂車的人，不然一定要找認識的業務才不會吃悶虧.. >_< ..
