自由取得Red Hat Enterprise Linux的功能 -- CentOS 3.4

之前我寫自由取得Red Hat Enterprise Linux的功能 -- White Box Enterprise Linux，就有提到同類型的社群還有CentOS。

她們網站的Slogan是
The Community ENTerprise Operating System，不曉得是不是暗指Whitebox是one man show :p

Anyway，為了裝OpenNMS，我之前在公司的lab裝過兩次Whitebox，不過下場都是被同事的正當用途要走，一台灌了Microsoft Windows 2000，一台灌了SuSE Linux。痛定思痛，這次我不再選擇高檔的機器，找了一台舊機器。CentOS套件更新較快，已經跟上了Red Hat Enterprise Linux 3 的 U4(Update 4)，所以我就乾脆跳船（算嗎？）到CentOS 3.4‧‧‧

為了要能夠執行指令yum update來更新套件，在我的環境必須要先設定proxy
# export http_proxy=http://proxy.my.company:port/
# yum update

另外，為了不要讓root可以遠端ssh登入，修改了
/etc/ssh/sshd_config
將
PermitRootLogin yes
改成
PermitRootLogin no

看來我還得先import GPG key 才行
# rpm --import http://mirror.caosity.org/centos-3/RPM-GPG-KEY-CentOS-3 --httpproxy proxy.my.company --httpport xxxx
（不需要透過proxy的話，請不要加上後面兩組參數）

另外，更新套件用
# yum -y update
就會自動下載和安裝，不用再按 y 了。
