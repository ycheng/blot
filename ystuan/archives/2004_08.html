[自由軟體介紹]檔案壓縮好幫手：7-Zip(1)

（本文原載於萬泰月刊2004年8月號，感謝作者Juan同意轉載。）

從這期開始，將會不定期地為大家介紹一些自由軟體。一般講到自由軟體可能會想到Linux這個作業系統下的應用程式，對使用者來說，自由軟體代表可以自由／免費地使用；對程式設計師來說這代表可以下載／修改程式原始碼。

自由軟體的這些好處，一定要用Linux之類的作業系統才能享受嗎？其實不論哪個作業系統，只要採用適當的授權條款來開發的軟體都是自由軟體，所以在Microsoft Windows底下也有很多唷，因為最近常有朋友詢問到檔案壓縮／解壓縮的問題，因此就先為大家介紹自由軟體裡最當紅的檔案壓縮／解壓縮的工具7-Zip。

講到檔案壓縮／解壓縮，壓縮有不同的演算法（格式），通常可用不同的附檔名（如zip，rar，gz）來區分。這次要介紹的7-Zip，有底下幾個特點：
1. 採用LGPL授權，即所謂不管個人或公司使用，完全免費！
2. 適用Microsoft Windows 98/ME/NT/2000/XP/2003
3. 安裝容易，圖形介面和指令列都可操作。
4. 支援下列格式：ZIP，CAB，RAR，ARJ，GZIP，BZIP2，TAR，CPIO，RPM，DEB，以及7z這個新的高壓縮率格式
5. 可產生.exe自動解壓縮檔案
6. 圖形介面內建41國語言（當然包含繁體中文囉～）

下載：目前最新版本是3.13版，可以直接連到這個網址下載：
http://download.ossacc.org/TM/ooo0515/software/7z313.exe。如果要檢查有沒有新版本，到http://www.7-zip.org/點選Download Now。在右側列出來的下載點中選擇一個：（圖一）

安裝的步驟也相當簡單：
1. 滑鼠在下載回來的7z313.exe點兩下(即開始執行)。
2. Do you want to install the 7-Zip 3.13? 選'Yes'
3. Which folder do you want to install to? 接受預設的C:\Program Files\7-Zip，直接按'確定'
4. 7-Zip 3.13 has been installed successfully.按”OK”
5. You must restart your computer before the new settings will take effect. Do you want to restart your computer now? 選Yes，讓電腦重新啟動。

更改語系：Windows左下角的”開始”->程式集->7-Zip->7-Zip File Manager，就可以看到主畫面。程式安裝好預設使用英文介面，要改變的話，在主畫面的下拉選單 Tools->Options->Language->從English改成需要的語系，例如Chinese Traditional（繁體中文）即可。（圖二）
