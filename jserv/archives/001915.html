演講：Linux 與消費性電子產品 (台北場)

月初的 blog [演講：Linux 與消費性電子產品] 提過應中正大學的邀約，給予一場關於 Linux 於手機、PDA 等消費性電子應用與思維變革的演講，而日前也有幸蒙 [國立政治大學資訊科學所] 給予演講邀約，因為上一場反應尚可，所以就延續舊題目繼續發揮，時間訂於下週一 (June 25)。不同於中正大學的場次 (大學部學生居多，所以臨時刪減了較深入的議題)，這次的對象主要是碩士在職專班的同學，所以我會在特定的技術項目著墨，諸如即時處理與進階能源管理。

以下是相關資訊：
  • 講題：Linux 與消費性電子產品
  • 時間：6/25 (一) 19:10～21:00
  • 地點：政治大學資科系 (大仁樓) 301教室
  • 講題簡介：
      Linux 的趨近成熟，對許多領域帶來了思維的變化，特別在消費性電子產品領域，原本採用的封閉式設計也受到挑戰，嵌入式的 Linux 裝置也逐漸在市場上嶄露頭角。然而，過去在個人電腦發展的 Linux 核心與相關的軟體，該如何適應消費性電子產品的重重考驗？本議程將以一個典型的 GSM 手機 / PDA 設計，探討採用 Linux 的動機與需求，並分析其因應之道。
  • 預定提綱：
    • 當 Linux 走出個人電腦
    • 平台的思維
    • SoC 加速 Linux 技術面的衝擊
    • 案例探討：Linux GSM 手機
這是小弟關於此議程的最後一場 talk，之後就不再作這類概括性的介紹，而是探索特定的技術項目，這也是 [Open RazzmatazZ Laboratory] (OrzLab) 最近在耕耘的技術細節，諸如透過系統模擬加速軟體開發、LOVER (Linux Optimized for Virtualization, Embedded, and Realtime)、嵌入式圖形與多國語文解決方案等等。

歡迎指教，謝謝！
