筆記型電腦嚴重毀損

三年多前來台北工作時，買了這台 HP nc4000 筆記型電腦，之後就常抱著它去許多場所，或在其上瘋狂 coding/hacking，賺取生活所需；或播放動人樂章，陪伴我挑燈夜戰讀完一本又一本厚厚的 spec 與 reference manual；或參與各式聚會，作野人獻曝之用；或作為最忠實的聽眾，隨內心種種思緒而舞動，轉化為一個個摻入感情的符號...。

就如許多人知曉的，這台筆記型電腦有 Ubuntu 創辦人 [Mark Shuttleworth] 的簽名 (詳見「Follow Your Dreams」一文)，不自覺這就跟社群活動有了高度關聯，利用閒暇，做了些針對自由軟體領域的心得分享，而在 2005 年成立自己的顧問諮詢公司 [jserv's lab]，更是常帶著這台筆記型電腦去客戶那邊解決棘手的技術問題或者給予教育訓練等。

說這台筆記型電腦是我這三年多的「親密愛人」，一點也不為過，不過我卻沒有好好珍惜。好幾次瘋狂 hacking GCC 時，還把系統操到過熱無法運作，之後還因大量的 regression test 讓磁區造成損壞。這之間還經歷多次的摔撞 (最著名的一次就算 [COSCUP遊記] 與 [COSCup 2006] 所提及的那樣)，屈指一算，保固期內送修了七次、更換了三次硬碟、兩次風扇，以及三次主機板，特別容易在炎炎夏日發生，或許也是因為居家的我，不願意開冷氣，也不吹電扇 (對於溫室效應的反動，以及對環境污染的自省)，更大幅降低了電腦的壽命。總之，這台筆記型電腦已經在昨天晚上宣佈停工，硬碟嚴重毀損，風扇也無法正確運作。

昨日給自己放了一天假，沈澱思緒，讀了近代史與練習素描，看著這「親密愛人」的殞落，是不捨，也是內疚。今日到辦公室尋求同事的協助，試著備份出硬碟中的資料，恰好瞥見 [檢測硬碟狀態 - smartmontools] 一文，只好苦笑回應，這幾個小時正用力的透過相關的 Linux 工具程式，對瀕死的硬碟作搶救，但仍回天乏術，還好特定的關鍵內容有保存。

這幾個月，正值自己對於工作、健康、精神生活樹立衝突、企求調適之時，總在對抗打擊身心狀況的病魔、作專案開發、顧問工作、撰稿寫作等項目中頻繁切換，看似很充實，其實卻讓自己陷入更加空虛的深淵。如今，連身邊的機器都不勝負荷，昨日損毀的筆記型電腦，與之前故障的伺服器 (作系統備份之用)，而，機械尚且如此，肉身之軀如我，怎能不警惕呢？

稍後我將購入新的筆記型電腦，不過既有的資料與專案或許不會悉數移轉，僅留下跟手邊工作或涉及客戶權益的智慧財產，停下腳步之際，我得更認真的面對未來的種種挑戰，也必須多作取捨，庸碌如我者，正尋求新的平衡狀態。安息吧，HP nc4000 筆記型電腦。

演講：Linux 與消費性電子產品 (台北場)

月初的 blog [演講：Linux 與消費性電子產品] 提過應中正大學的邀約，給予一場關於 Linux 於手機、PDA 等消費性電子應用與思維變革的演講，而日前也有幸蒙 [國立政治大學資訊科學所] 給予演講邀約，因為上一場反應尚可，所以就延續舊題目繼續發揮，時間訂於下週一 (June 25)。不同於中正大學的場次 (大學部學生居多，所以臨時刪減了較深入的議題)，這次的對象主要是碩士在職專班的同學，所以我會在特定的技術項目著墨，諸如即時處理與進階能源管理。

以下是相關資訊：
  • 講題：Linux 與消費性電子產品
  • 時間：6/25 (一) 19:10～21:00
  • 地點：政治大學資科系 (大仁樓) 301教室
  • 講題簡介：
      Linux 的趨近成熟，對許多領域帶來了思維的變化，特別在消費性電子產品領域，原本採用的封閉式設計也受到挑戰，嵌入式的 Linux 裝置也逐漸在市場上嶄露頭角。然而，過去在個人電腦發展的 Linux 核心與相關的軟體，該如何適應消費性電子產品的重重考驗？本議程將以一個典型的 GSM 手機 / PDA 設計，探討採用 Linux 的動機與需求，並分析其因應之道。
  • 預定提綱：
    • 當 Linux 走出個人電腦
    • 平台的思維
    • SoC 加速 Linux 技術面的衝擊
    • 案例探討：Linux GSM 手機
這是小弟關於此議程的最後一場 talk，之後就不再作這類概括性的介紹，而是探索特定的技術項目，這也是 [Open RazzmatazZ Laboratory] (OrzLab) 最近在耕耘的技術細節，諸如透過系統模擬加速軟體開發、LOVER (Linux Optimized for Virtualization, Embedded, and Realtime)、嵌入式圖形與多國語文解決方案等等。

歡迎指教，謝謝！
