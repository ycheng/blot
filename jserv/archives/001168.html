透過 MIT roofnet Project 快速架設社區無線網路

剛剛在 moto.dot 閱讀 jthchang 撰寫的 [MIT roofnet project快速架設社區無線網路]，立刻吸引我的高度關注，以下引述部分內容：
    「這是幾個麻省理工學院學生所做的研究題目，目前已在波士頓的某個公寓區架起了一個由 50 幾個點所組成的網路 (They have an experimental multi-hop 802.11b mesh network consisting of about 50 nodes in apartments in Cambridge, MA，詳見 [CommunityInternet.us])，還有一個每五分鐘更新一次的網路狀況圖 (詳見 [Roofnet Real-Time Link Map])。目前架設一個 node 最簡單的方法就是用Netgear WGT634U USB firewall router (詳見 [modified wgt634u])，很特殊的是，這是一台基於 Debian 所發展的 embedded Linux system。MIT rootnet 之所以可以快速架設網路原因在於每一個 wireless node 都有自我設定和自我修復的功能 (self-configuring and self-healing，詳見 [MIT Roofnet Implementation] 與 [Champaign-Urbana Community Wireless Network | CUWIN])，大意就是插上電以後什麼都不用管，連阿公阿媽也會架，有興趣架設社區無線網路的朋友可以參考參考。」
