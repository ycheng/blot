迴避 spam comment

今天 caleb 在 #dot 提到要輸入帳號密碼才能瀏覽 comment，這是昨天改的迴避 spam comment 的方式，當處理 comment 的 CGI 被呼叫時，瀏覽器會跳出要求認證的視窗，如下：

User Name 為："nospam"，而 Password 為："iamnotspam"，因為 spam 攻擊實在太嚴重，才會出此下策，造成您的不便，請多包涵，謝謝！
