2005 年摩托學園/Debian User 聚會

紀錄人： Jim Huang
* 按：以下的圖片 click 後可得全圖

去年的 [摩托學園第一次聚會] 相當成功，也遇到很多新朋友，當然最興奮的是能夠見到過去在網路上耳聞的前輩的樣貌。去年在聚會結束後，我寫了一份紀錄 [2004 年摩托學園/Debian User 第一次聚會]，看來今年也不能免俗，我就稍微紀錄一下，詳細聚會資訊請參考 [MotoWeb2005Meeting]，本次聚會活動錄影是由 Jesse 提供，可參閱 [2005 摩托學園聚會]，雖然 Jesse 在他的 blog [偏暗的照片] 提到因為是在室內(地下室)，全程使用閃燈，所以照片偏暗，不過整體來說拍得還很好，感謝 Jesse 熱心支援。


週六 (May 7, 2005) 中午用餐後，我就騎著 GIANT 腳踏車慢慢的從內湖的金湖路移動到新生南路一段，因為剛用餐完畢，不方便飆太快，所以是悠閒地踩著踏板，就這樣，大約花了一小時又十分鐘才到。到了 OpenStudio 後，才一坐下來，就忙著跟許多熟面孔打招呼，比方說 moto、AceLan、faifai、Jack、hata、chihchun、Jesse，以及 FourDollars。

在正式開始前，moto 分享了幾張賢伉儷去澳洲旅遊的照片，提到澳洲，大家一定會想到袋鼠與無尾熊，當然 moto 也捕捉了許多合照，甚至還有小布 (好神秘？) 與這些可愛動物的合照。moto 還提到有種澳洲獨有的動物，就稱為「王八」，後來 turtle5001tw (醫生前輩) 就補充說這是一種巨型的老鼠，哇，真有意思，原來「鼠輩」也可以長得這麼大隻。

在幾位朋友陸續趕來後，重頭戲就開始了，當然要自我介紹一番，這次我遇到的幾位新朋友有：PCMan (是阿，就是傳說中的 PCMan X 作者阿)、hakken (據說這 nickname 是來自狗名？)、timwu (moto 的前同事)、Mat (其實在上次的 [中文輸入自由軟體工作坊] 就有碰面，但我太健忘，第二次見面才把 "Mat" 這個 nick 跟本人建立關聯)，還有一位是 AceLan 的學弟，不過 nick 忘記寫下來。

又過了一段時間，AndrewLee、Arne、kanru、palmpilote 與 descent 陸續趕到，只見 Arne 一趕上，就開始分享他的 [CJKUnifonts in Debian]，字型處理本身就是相當複雜的議題，扯上中文又更複雜了，Arne 來自德國，基於對於中國文化的愛好，開始填補 Unicode TrueType 字型中欠缺的中文符號，整個簡報過程，台下不停的讚嘆聲傳出，實在是太偉大了，記得我之前寫信 (用傳統紙筆) 給 jie，因為寫到一半忘記忘記字型的書寫方式，只好把信紙扔掉，改用英文書寫，看到 Arne 的投入，讓我覺得好羞愧。

在一陣寒暄後，換我的 [jserv's lab] 展示幾個之前做的玩具：
  • Xorz (詳情可參考 blog [Xorz 公開了])
  • 新酷音 with SCIM / IIIMF
  • Linetconf (詳情可參考 blog [Linetconf (Linux Networking Configurator)])
  • Xorzzz Window Manager
  • Eclipse/Kaffe (詳情可參考 blog [Eclipse 3.0.1 倩影])

上圖就是將著名的整合開發平台 [Eclipse] 在 Kaffe VM 上運作的展示畫面 (詳情可查看全圖檢閱 Runtime 屬性)，效果還不錯，可惜似乎發現 Kaffe 出了點小問題 (在展示前才剛 commit 一個小 patch，所以...)，造成 Eclipse 的使用有點緩慢，不過基本使用應該沒有大問題。

在進行下個活動前，我們閒聊一段時間，當然不免提到最近的工作，聊些八卦，之後就是 chihchun 分享他們之前去北京參加 Asia Debian Aisa Debian Mini-Conf 2005 的經驗，雖然在 moto.debian.org.tw 上面已經分享過 [Asia Debian Aisa Debian Mini-Conf 2005]，不過 chihchun 口頭敘述實在有趣的多，比方說 [這張照片] 中，那位仁兄津津有味的享用串燒，而事實上他正在 De"Bug" 喔，對的，就是把 "Bug" 給... :P 。 chihchun 也針對 Debian@Taiwan 的現況做了一份詳實的報告，比方說年度計畫、經費使用，還有待作事項，至於大家比較關心的經費議題，稍後會公佈募款與硬體捐贈的方式。

後來，Jouston 終於趕上聚會，為我們分享 [ 如何參與 DWN 中文翻譯]，同時我們也必須對這些持續翻譯 DWN (Debian Weekly News) 的幕後英雄給予掌聲，當然有新血的投入與支持是再好不過了。

最後，就是有獎徵答了，我捐了一本高中開始閱讀的 Linux Programming Guide、Planex GW-NS54GM 無線網路卡 (驅動方式請參閱 [Planex GW-NS54GM 無線網路卡驅動攻略 (2)]，以及小企鵝玩偶，雖然我提出的問題有點艱難，不過與會的朋友還是用力的找出答案，恭喜 AceLan、Jouston，以及 FourDollars 獲得獎品。

聚會尾聲，又是一番閒聊，看來 X Window System 眾多的嶄新技術讓許多朋友感到高度的興趣，之前在 IRC 上就有頗多朋友提出他們的疑惑，不過還是有待理解的項目，所以今年的 IRC Conference 有可能會針對這些議題再做介紹，當然啦，也可以慢慢等我的新書現身啦。

會後，我們七個人 (AndrewLee、Arne、chihchun、kanru、FourDollars、descent、jserv) 相約去師大路上的一家印度餐廳用餐，這家的調味很不錯，口味也比較豐富些，最重要的是，上菜快多了。於是我們又聊了許多新話題，Arne 也提到 CJK Unicode Fonts 與 Pango 的議題，不過這對於我還是有點困難，嗯，先記錄下來，等比較有空再來研究。

當然啦，天下沒有不散的筵席，大約在晚上八點，一行人就各自道別，而我也騎著腳踏車，慢慢回到內湖，大約花了一小時。Ok，咱們有機會再見面了。
