ZTerm : 以 Java 撰寫的 VT100 Telnet/SSH 終端機

[ZTerm] 一開始是個專為 BBS 設計的連線軟體，以 Java 撰寫，模擬 VT100 終端機，支援 Telnet 與 SSH2，提供分頁 (Tab) 功能，其他 BBS 軟體常見的彩色複製、貼上也有提供，而為了邪惡的 Big5 日文假字，也在內部實做了轉換表格，依據其 source code 的 [LICENSE] 顯示，應該是以 GNU GPLv2 發行，不過網頁沒有很清楚的寫出來。
