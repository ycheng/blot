不喜歡用 Movable Type

首先，要感謝 AceLan 與 PowerOp 兄的協助，我才能在 blog.linux.org.tw 上撒野，然後也要感謝 hcchien 告知 Jedi 的大作 [Movable Type 文件集]，我才對 MT (Movable Type) 有了進一步的認知。可是我還是有很多疑惑，比方說內嵌 HTML tag 時，呈現的行為總是有點怪：

2. ＜pre＞...＜/pre＞ 呈現的效果也是跑出一堆無用的 ＜br＞
看起來真的很不舒服 :(

到底要怎麼解？
