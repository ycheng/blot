「Applied Computer Science Concepts in Android」簡報上線

2010 年最後一日，應台灣大學資訊工程系的邀請，小弟給予一場名為 "Applied Computer Science Concepts in Android" 的演講，詳情可參考 [演講公告]。題目的靈感來自《Applied Operating System Concepts》一書，該書以 Java 為探討對象，談論諸多作業系統的原理，兼顧理論與實務，而在現實中，Android Open Source Project 就是再好不過的研究對象，廣泛包含通訊、圖形、資料庫等等系統。

本議程採用 Android 作分析對象，挑選若干軟體元件來驗證電腦科學的概念，尤其是一般人較難聯想到的地方，如藏在 2D/3D 圖形系統的動態編譯器/組譯器，與嵌入於資料庫系統的虛擬機器。提綱如下：
  • Android Internals
  • Compiler: 2D/3D Graphics
  • OS: Telephony
  • VM: Database
