RSS Reader Panel for Mozilla Firebird

今天在我的 Mozilla Firebird 中裝上了『RSS Reader Pannel Extension』, 不是什麼新東西了, 不過一試之下才知道好用哩... :D

在 "Tools" -> "Options..." -> "Extensions" 中按一下 "Get New Extentions", 就可以它, 按下網頁上的 Install 就可以安裝囉!

安裝好之後, 重開 Firebird, 然後在 Bookmarks 中新增一個 Folder, 把你想看的 RSS 的 url 都 bookmark 到這個 Folder 中, 然後設定 RSS Reader Panel, 把 RSS List Folder 指到這個 Folder 來, 搞定!

怎麼用呢? 選 "View" -> "Sidebar" ->"RSS REader Panel", 就可以看到左邊多了一個 frame, 選你想看的 RSS 就可以在該 frame 的下方看到內容。如果想看詳細內容, 可以把 RSS Reader 的 "View" -> "Open In Contens Area" 勾起來, 這樣就可以在右邊的 frame 中閱讀詳細內容了。附上 screenshot 一枚...
